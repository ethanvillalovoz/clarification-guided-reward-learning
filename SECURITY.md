# Security Policy

## Reporting

Use a private GitHub security advisory for vulnerabilities. Do not include participant data, private correspondence, credentials, or unreleased research materials in public issues.

## Data Boundary

The version 1.1 simulation uses no participant data, network services, model weights, or secrets. Generated traces contain only hand-authored toy hypotheses and synthetic object labels.

Any future participant study or robot deployment needs a separate data-management plan, consent process, access controls, retention policy, and institutional review before data enters this repository.
